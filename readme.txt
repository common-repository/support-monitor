A Simple Support Monitoring Tool for WordPress.

== Description ==

Support Monitor will help you find users support topics that are unresolved based on time limit, Monitor specific plugin or multiple at a time

= Links =

* [Github project page](https://github.com/emtiazzahid/wp-support-monitor)

== Installation ==

If installing the plugin from wordpress.org:

1. Upload the entire `/support-monitor` directory to the `/wp-content/plugins/` directory.
2. Activate Support Monitor Plugin through the 'Plugins' menu in WordPress.
3. Enjoy.

== Frequently Asked Questions ==

= What can I do with this Support Monitor plugin? =

Suppose you have multiple WordPress plugin, and you want to monitor those plugin support topics
with that plugin you can check if their any unresolved/not replied topics withing a time range
